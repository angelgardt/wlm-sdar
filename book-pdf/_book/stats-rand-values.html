23.4.6 Нормальное распределение

\[ X \sim \mathcal{N}(\mu, \sigma^2) \]

\[ f(x) = \frac{1}{\sigma \sqrt{2\pi}}e^{-\frac{1}{2}\big(\frac{x - \mu}{\sigma}\big)^2} \]

Более того, известны вероятности, с которыми значения случайной величины, распределенной по нормальному закону, попадают в определенные интервалы стандартных отклонений (Рисунок 23.1):

\[ \mathbb{P}\big( X \in (\mu\!-\!\sigma, \mu\!+\!\sigma) \big) = 0.682 \]

\[ \mathbb{P}\big( X \in (\mu\!-\!2\sigma, \mu\!+\!2\sigma) \big) = 0.954 \]

\[ \mathbb{P}\big( X \in (\mu\!-\!3\sigma, \mu\!+\!3\sigma) \big) = 0.996 \]

Задавая различные параметры распределения (математическое ожидание и дисперсию), мы можем получать разные нормальные распределения (Рисунок 23.2).
